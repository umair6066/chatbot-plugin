Products — JSON file URL

Pass a productsUrl and the widget fetches the file at runtime. Update products.json anytime — no code change needed.

Try asking: "what products do you have?" · "show prices" · "what's in stock?"

⚠ Requires a server for the URL fetch to work. Browsers block fetch() when a file is opened from disk.

Run: npx serve samples/ then open http://localhost:3000/with-products-url.html

Opening from disk still shows the chat widget — it just falls back to the inline products below.
