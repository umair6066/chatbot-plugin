Products — JSON file URL

Pass a productsUrl and the widget fetches the file at runtime. Update products.json anytime — no code change needed.

⚠ Requires a server for the URL fetch to work. Browsers block fetch() when opened from disk.

Run: npx serve samples/ then open http://localhost:3000/with-products-url.html

Opening from disk still shows the chat — it falls back to the inline products below.

Try asking: "what products do you have?"  ·  "show me prices"  ·  "what's in stock?"  ·  "cheapest option"

Suggestion chips with URL products

When products load from a URL the widget refreshes the welcome message chips automatically. The chips update from generic defaults to product-specific ones once the file is fetched.

Custom chips alongside URL products

<script>
  ChatbotWidget.init({
    productsUrl: "./products.json",
    // Override auto-generated chips with your own
    suggestions: [
      "What products do you have?",
      "What's your return policy?",
      "Do you have free shipping?",
      "Contact support",
    ],
  });
</script>

Custom suggestions take priority — the auto-refresh on URL load is skipped.
