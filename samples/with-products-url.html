Products — JSON file URL

Pass a productsUrl and the widget fetches the file at runtime. Update products.json anytime — no code change needed.

Try asking: "what products do you have?" · "show prices" · "what's in stock?"

⚠️ Requires a server. Browsers block file fetches when opening HTML directly from disk (file://). Run a local server first:

npx serve samples/  or  python3 -m http.server 8080

Then open http://localhost:8080/with-products-url.html
