Products — JSON file URL

Pass a productsUrl and the widget fetches the file at runtime. Update products.json anytime — no code change needed.

Try asking: "what products do you have?" · "show prices" · "what's in stock?"

This page detects whether it is running on a server or opened directly from disk.

• On a server — loads products from ./products.json via URL.
• Opened as a file — uses an inline fallback array automatically (browsers block fetch() on file://).
